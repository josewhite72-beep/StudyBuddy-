# 📚 StudyBuddy

**English language learning platform for PreK–Grade 6 students in Panama.**

StudyBuddy is an offline-first Progressive Web App (PWA) designed for low-connectivity classroom environments. Students practice Listening, Reading, Speaking, Writing, and Mediation skills through scenario-based interactive activities aligned to MEDUCA standards and CEFR levels.

---

## Features

- PIN-based student login (no email required)
- Scenario-based curriculum — 6 scenarios per grade, 5 skills each
- Star progression system (1–3 stars per activity)
- Performance-based scenario unlocking (80% average required)
- Teacher dashboard with per-grade analytics and manual unlock
- Works offline after first load
- Installable as PWA on Android devices

## Grades & CEFR Levels

| Grade | CEFR |
|-------|------|
| Grade 3 | A1.1 |
| Grade 4 | A1.2 |
| Grade 5 | A1.3 |
| Grade 6 | A2.1 |

## Tech Stack

- Vanilla HTML/CSS/JS (single file, no build step)
- Supabase (PostgreSQL backend + RPC functions)
- Offline-first architecture with localStorage session
